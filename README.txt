##Background
This repo contains all the files needed to complete the final project for the Getting and Cleaning Data course

##Dataset
The dataset for this project can be found here: [http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones]

##Files in this repo
- code book.md <- this contains a description of the steps taken to clean the data as well as an explanation of all the variables used
- analysis.R <- this contains the R scipt writted to clean the data
- final_data.txt <- this contains the final dataset created
